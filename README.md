# LeadAnalytics — Парсер и Лидогенератор для Начинающих Аналитиков Данных

Этот инструмент разработан специально для тех, кто хочет вкатиться в сферу **Data Science и Data Analytics**, набрать реальный коммерческий опыт на фрилансе и сформировать сильное портфолио, работая с реальным бизнесом (кофейни, пекарни, салоны красоты, кондитерские и т.д.).

Парсер собирает заведения по выбранному городу (например, Москва) с контактными данными и автоматически оценивает их **аналитический потенциал** — насколько бизнесу нужна аналитика и насколько легко с ним связаться.

---

## 🚀 Как запустить приложение

1. Убедитесь, что у вас установлены все зависимости из `requirements.txt`:
   ```bash
   pip install -r requirements.txt
   ```

2. Запустите веб-сервер:
   ```bash
   uvicorn main:app --reload
   ```

3. Откройте браузер по адресу:
   [http://127.0.0.1:8000](http://127.0.0.1:8000)

## 📈 Методология: Как это поможет получить первый опыт и работу

### 1. Каких клиентов искать?
Парсер автоматически делит заведения на группы по **Аналитическому потенциалу**:
- **HIGH (Высокий):** Независимые одиночные заведения, у которых есть сайт и телефон. У них почти наверняка установлена система учета (iiko, r-keeper, YClients, МойСклад), в которой копятся данные. При этом у них нет денег на штатного Data Analyst. Они — ваша главная цель.
- **MEDIUM (Средний):** Есть только один канал связи (или сайт, или телефон). С ними сложнее связаться, но они тоже подходят.
- **LOW (Низкий):** Крупные сети (Cofix, Шоколадница и т.д.) — у них есть свои IT-отделы и CRM-специалисты, новичков они не возьмут. Или заведения без каких-либо контактов на карте.

### 2. Как связаться с заведением?
1. **Онлайн:** Выберите заведение в таблице с высокой оценкой потенциала. Программа сгенерирует для вас готовое письмо (питч) на русском языке. Скопируйте его и отправьте через форму обратной связи на сайте заведения или на email.
2. **Офлайн (самый эффективный способ):**
   - Найдите на интерактивной карте в приложении кофейню рядом с вашим домом/учебой.
   - Сходите туда, закажите кофе.
   - Спросите у бариста, можно ли поговорить с управляющим или владельцем.
   - Расскажите, что вы начинающий аналитик, живете неподалеку и хотите **бесплатно** помочь им оптимизировать списания выпечки или настроить аналитику меню в обмен на отзыв и кейс в портфолио.

### 3. Какую систему учета они используют?
- **Кофейни/Кафе/Рестораны:** iiko, r-keeper, Poster, Quick Resto, 1С:Общепит.
- **Салоны красоты:** YClients, Altegio, Арника.
- **Маленький ритейл/Цветы:** МойСклад, 1С:Розница.

### 4. Какие данные просить?
Вам не нужен доступ к их аккаунтам! Попросите владельца выгрузить простые файлы в формате Excel/CSV:
- **Для общепита:** Выгрузка чеков (транзакций) за последние 3-6 месяцев (дата, время, номер чека, название блюда, количество, цена, себестоимость).
- **Для услуг:** Журнал записей клиентов за последние 3-6 месяцев (анонимизированный ID клиента, дата записи, название услуги, мастер, статус визита, стоимость).

### 5. Что делать с данными (Кейсы в портфолио):
1. **ABC/XYZ-анализ меню:** Покажите владельцу, какие позиции приносят 80% прибыли (класс А), а какие лежат мертвым грузом и списываются.
2. **Прогнозирование спроса:** Постройте простую модель (даже в Excel или на Python с помощью библиотеки `statsmodels` / `prophet`), которая предсказывает продажи на следующую неделю в зависимости от дня недели и погоды. Это поможет владельцу заказывать ровно столько молока и выпечки, сколько купят, снизив списания на 15-30%.
3. **RFM-анализ лояльности:** Разделите клиентов по давности визитов (Recency), частоте (Frequency) и потраченным деньгам (Monetary). Владелец сможет сделать точечную рассылку (например, дать скидку тем, кто раньше ходил часто, но не был уже месяц).
4. **Интерактивный дашборд:** Создайте красивый дашборд в **Yandex DataLens** (это бесплатный российский BI-инструмент) или Tableau/Power BI и отдайте его владельцу.

После выполнения проекта попросите у владельца письменный отзыв на бланке или в Telegram. Этот кейс с графиками, описанием вашей работы и отзывом реального бизнеса на 100% выделит вас среди сотен других кандидатов на позицию Junior Data Analyst!
